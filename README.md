Piano:bit
=========

Yahboom Piano:bit (without neopixel) for use with Bluetooth MIDI


Example
-------

 * [Shared project](https://makecode.microbit.org/_4YEHDuF4tUDm)
 * [Video](https://peertube.social/videos/watch/3e6eea51-5c22-45f4-90e8-24a727166a8d)


Pinout
------

| Pin | Micro:bit |
|-----|-----------|
| VCC | 3V        |
| SCL | P19 (SCL) |
| SDA | P20 (SDA) |
| IRQ | P2        |
| LED | P1        |
| FM  | P0        |
| GND | GND       |
